実行時に RunnableConfigで設定できる。

```py
workflow = StateGraph(MessagesState)

...

app = workflow.compile()

app.stream(inputs, config={"run_name": "ExampleAgent", "metadata": {"tenant": "t1"}, "project_name": "t1"})
```

- run_name
- metadata
